Contents
--------
This folder contains the input images used to test the scanner pipeline.
All images were found or captured manually (no programmatic generation).

Image sources:
  - Photos taken with a smartphone (real product barcodes, QR codes)
  - Screenshots of QR codes displayed on screens
  - Downloaded sample images (libre/open licence)

Code types present:
  - QRCODE   : square matrix codes (URL, text, JSON payloads)
  - CODE128  : high-density linear barcode
  - EAN-13   : standard 13-digit retail product barcode
  - EAN-8    : compact 8-digit retail barcode

Output files (generated by the pipeline):
  - result_*.png         : annotated copies of each input image
  - pipeline_detail.png  : 6-panel preprocessing visualisation
  - evaluation_chart.png : bar chart of detections per image
